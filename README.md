Команды терминала

- Открыть сервер:
  heroku open

- Открыть логи:
  heroku logs --tail

- Запуск локально без deploy:

  - установить: npm i nodemon --save-dev
  - вписать: "scripts": {"develop": "nodemo index"}
  - запустить: npm run develop

- Запушить на сервер:
  git push heroku main
